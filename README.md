# Spring-TeamProject-PetPlaza
## 🖥 프로젝트 소개
기존 업체들의 높은 이용가격에 부담을 느낄 소상공인을 위한 쇼핑몰 통합관리시스템입니다.



## ⏰ 개발 기간
2023.10.20 - 2023.11.20

## 👫 멤버 구성
- 팀장 정희락: 거래처관리 | 발주관리
- 팀원 고연진: 로그인 | 매출관리 | 일정관리
- 팀원 황기준: 제품관리
- 팀원 이진형: 주문관리
- 팀원 한승우: 회원관리
- 팀원 박상빈: 거래처단가관리 | 재고기록관리

## ⚙ 개발 환경
- IDE: intelliJ 
- springboot: v2.7.17
- node: v20.8.1
- 협업툴: Jira ,Git-hub

## ⌨ 사용 기술
- Java
- JavaScript
- Html/Css
- Axios
- MySql
- spring
- React
- JPA
